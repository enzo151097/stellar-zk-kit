Stellar ZK Kit

A local developer toolkit that turns a Noir circuit configuration into a Stellar-ready zero knowledge proof pipeline.

What It Generates

  • Typed Soroban verifier wrapper contract.
  • TypeScript proof encoding helpers.
  • React/Vite evidence dashboard.
  • Mutation tests for public input safety.

Why It Matters

The kit removes common ZK integration mistakes: mismatched public input order, endianness issues, and complex proof byte packing.

Architecture

zkkit.toml → Noir + bb → zkkit build → Stellar

Demo Circuit

proof_of_funds proves a private balance satisfies a public minimum, then shows local proof output and verified Stellar testnet evidence.
